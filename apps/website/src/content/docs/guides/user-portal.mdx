The user portal intentionally exposes fewer controls than the administrator
interface. A user can manage only their own account and VPN devices.

## VPN dashboard

The dashboard summarizes:

- total upload and download usage;
- quota consumed and remaining;
- account expiration or unlimited duration;
- accumulated connection time;
- active connections and recent device activity;
- the account speed policy.

Usage is aggregated from Xray counters. NoxRouteNeo does not store requested
domains, URLs or browsing history.

## Account settings

Users can change their password, inspect active web sessions, revoke sessions,
enable TOTP and select a personal interface language. The personal language does
not change the default for other accounts.

<Aside type="note" title="Web sessions and VPN connections are different">
  An active web session means a browser is signed in to the portal. Device VPN
  activity is shown separately and is based on Xray and traffic-gateway
  telemetry.
</Aside>
