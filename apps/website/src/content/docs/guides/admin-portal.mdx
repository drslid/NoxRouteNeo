## Account policies

An administrator can set these values per VPN user:

- duration in days, or unlimited;
- transfer quota in GB, or unlimited;
- maximum registered devices;
- TCP speed limit;
- enabled or suspended state.

Routine account and device changes are synchronized through Xray's local API so
existing tunnels do not need a full Xray restart.

## Operational boundaries

- PostgreSQL, Xray's API and internal service APIs remain private.
- Security events and audit history are operational metadata, not browsing history.
- The flow ceiling is a safety admission limit, not the number of users the VPS can support.
- CPU, memory and network throughput normally become limiting before the configured flow ceiling.

<Aside type="tip" title="Keep owner operations separate">
  Only the owner should create administrators or perform sensitive administrator
  password resets. Daily user management can be delegated to an administrator
  account with narrower permissions.
</Aside>
